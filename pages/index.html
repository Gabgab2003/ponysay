ponysay as an awesome program to displays ponies giving messages in your terminal.
It has a lot of features; you can use its info manual to explore it, the manual is also
available [here] or [here] as an PDF. To the left you will found some recommended
sections of the manual to read if you are not up for reading the entire manual.

If you cannot find a package distribute for your OS, take a look [here] or build it yourself.
On most systems with the base packages, base-devel packages and Python 3 installed,
downloading the software and running sudo ./setup.py --everything install
should install ponysay, with everything it has to offer. If you are not privileged to run as
super user you can install install it for just your user by omitting sudo and adding
--private.

If you are missing your favourite pony, [ask] and I we will add it (as soon as possible).
