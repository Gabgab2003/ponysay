9.2 Reporting bugs

If you find a bug in ponysay, install the last version from https://github.com/erkin/ponysay, and if it is still present, please report it at https://github.com/erkin/ponysay/issues. Please be as descriptive as possible, as it will help us verify it solve it faster.
