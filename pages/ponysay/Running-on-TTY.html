3.4 Running on TTY

If you use TTY and have a custom colour palette, you should also add to your ~/.bashrc, before fortune | ponysay: 

     [ "$TERM" = "linux" ] &&
         function ponysay
         {   exec ponysay "$@"
             #RESET PALETTE HERE
         }

You should read more about this in KMS ponies.
