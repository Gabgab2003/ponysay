11.4 Uninstalling

If you did not install ponysay with a package manager, but rather manually from the upstream, you can uninstall it by running make uninstall.

Well written package manages will uninstall files that the package is no longer using, i.e. if deleted, moved or renamed. To uninstall files that are not longer used, by the currently installed version you will need that versions Makefile. To perform an uninstallation of old files run make uninstall-old.
